## Description

User account and jwt authorization strategy with Nestjs and GraphQL

## Installation

```bash
$ yarn install
```

## Running the app

```bash
# development local
$ yarn d

# development docker
$ npx docker-compose up
```

## Test

```bash
# unit tests
$ yarn t

# unit tests watch mode
$ yarn tw

# unit tests with coverage
$ yarn tc
```
